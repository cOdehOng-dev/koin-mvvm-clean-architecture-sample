# koin-mvvm-clean-architecture-sample

## Description
```
클린 아키텍처를 바탕으로 의존성 주입(DI) 라이브러리 koin을 사용한 MVVM 샘플 프로젝트
```

## Spec
- MVVM Architecture Pattern
- AAC(ViewModel, LiveData, Room)
- Repository Pattern
- Clean Architecture
- Databinding
- RxJava
- Coroutine
- Flow

## Library
- Retrofit
- Okhttp
- Glide
